# Baby Jane Magros Rodriguez — VA Portfolio

A clean, single-page portfolio website ready for GitHub Pages.

## 📁 Folder Structure

```
babyjane-portfolio/
├── index.html        ← main website file
├── images/
│   ├── hero.jpg      ← YOUR PHOTO (hero section, right side)
│   └── about.jpg     ← YOUR PHOTO (why section, left side)
└── README.md
```

## 🖼️ Adding Your Photos

Before publishing, add your photos to the `images/` folder:

1. **`images/hero.jpg`** — Used in the hero section (right side, tall portrait)
   - Recommended: portrait orientation, at least 900×1200px
   - Subject should be centered/top-aligned

2. **`images/about.jpg`** — Used in the "Why Choose Me" section
   - Recommended: portrait or square, at least 600×600px
   - Can be the same photo or a different one

> You can use the same photo for both — just save a copy with each filename.

## 🚀 Publishing to GitHub Pages

1. Create a new GitHub repository (e.g. `babyjane-portfolio`)
2. Upload all files keeping the folder structure intact
3. Go to **Settings → Pages**
4. Under "Source", select **main branch / root**
5. Click Save — your site will be live at:
   `https://YOUR-USERNAME.github.io/babyjane-portfolio/`
